# 📝 To-Do App

## 📌 Description

This is a simple and interactive To-Do List web application built using JavaScript.
It allows users to manage their daily tasks efficiently by adding, completing, and deleting tasks.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🎨 Visual feedback for completed tasks

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
